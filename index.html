Plataforma B2C

Descreve interesse de carreira / Seleciona vagas de interesse

O candidato manifesta interesse em determinadas oportunidades, permitindo que a IA evolua o Emprecard de "quem eu sou" para "onde quero chegar" e sugira uma trilha de capacitação mais precisa para a conquista dessas vagas.

Plataforma B2C

Recebe recomendações das trilhas (Studion)

Com base no perfil criado, o candidato recebe recomendações personalizadas de trilhas de capacitação disponíveis no Studion.

Plataforma B2C

Integra cadastro para acesso no Studion

Com base no e-mail e CPF, o sistema consulta o Studion para verificar se o usuário já possui cadastro no Capacita Coop e, quando possível, realiza o registro automático para que ele acesse a trilha de capacitação diretamente.

Studion

Inicia trilha de capacitação no Studion

O candidato é direcionado diretamente para o primeiro curso da trilha para iniciar a capacitação, com acompanhamento e engajamento ao longo do processo pelo Studion.

Studion

Conclui e é aprovado na trilha

A plataforma EmpregaCoop recebe, via API, a confirmação de conclusão da trilha de capacitação.
